# 🧠 Awesome ML Papers Weekly

Еженедельные подборки лучших статей с arxiv по Machine Learning, отобранные вручную.

Курирует: [Лора Нейронова](https://t.me/lora_neuro) — ML researcher, PhD candidate.

## Зачем это

На arxiv выходит 100+ статей в день по ML/AI. Читать всё невозможно. Я трачу 10-15 часов в неделю на чтение и выбираю 3-5 статей которые реально стоят внимания.

## Критерии отбора

- 🔬 Новая идея, а не +0.3% на бенчмарке
- 💡 Меняет мышление или практику
- ✅ Экспериментально обоснована
- 🗣️ Я могу объяснить суть за 2 минуты
