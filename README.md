# 🧠 Awesome ML Papers Weekly

Curated weekly picks of the best ML/AI papers from arxiv. Hand-selected, briefly explained.

Curator: [Lora Neironova](https://t.me/lora_neuro) — ML researcher, PhD candidate @ DeepMind.

> На русском: еженедельные подборки лучших статей с arxiv по ML, отобранные вручную.

## Why this exists

100+ ML papers hit arxiv daily. Most are incremental. I spend 10-15 hours/week reading and pick 3-5 that actually matter.

## Selection criteria

- 🔬 Novel idea, not just +0.3% on a benchmark
- 💡 Changes how you think or code
- ✅ Experimentally grounded
- 🗣️ I can explain the core idea in 2 minutes
